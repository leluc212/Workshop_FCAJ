Advanced Analytics with Athena & AI-Powered Serverless Security

AWS Security Analytics System

I. Overview

Hệ thống phân tích log từ AWS WAF nhằm phát hiện tấn công web hiệu quả và tối ưu chi phí. Dữ liệu được lưu trên S3 và chuyển sang Parquet để giảm chi phí Athena.

  • Layer 1: Athena lọc request nghi ngờ
  • Layer 2: AI (Bedrock) giảm false positive
  • Event-driven + threshold-based → tiết kiệm chi phí

II. Architecture

  • Edge & Security Layer
  • Processing & Cataloging Layer
  • Analysis & Reporting Layer
  • Report & Notification Layer

Cost Optimization High Accuracy Serverless

III. Edge & Security Layer

  • CloudFront: phân phối nội dung
  • AWS WAF: bảo vệ (SQLi, XSS, rate limit)
  • Log lưu để phân tích

IV. Processing & Cataloging

Storage

  • S3_rawlog (JSON)
  • S3_parquet (optimized)

Processing

  • Lambda: JSON → Parquet
  • Partition theo thời gian

Catalog

  • Glue Crawler → tạo schema
  • Glue Data Catalog → metadata

Query

  • Athena → filter request nghi ngờ

V. Event-driven Query

  • Trigger từ S3 hoặc Lambda
  • Lambda gửi query Athena

VI. Analysis & Reporting

Preprocessing

  • Chuẩn hóa dữ liệu
  • Aggregate (IP, pattern...)

AI Invocation

  • Chỉ gọi AI khi vượt threshold

AI Analysis

  • True attack / False positive
  • Risk level + explanation

Decision

  • Alert / Save / Ignore
  • Lưu vào S3_AI_Report

VII. Notification

  • Lambda tổng hợp report
  • SNS gửi alert

VIII. Benefits

  • Chi phí thấp (Athena + Parquet)
  • Hiệu năng cao
  • Độ chính xác cao
  • Serverless

IX. Cleanup

  • Xóa Glue, Lambda, SNS, IAM
  • Xóa S3 bucket
  • Xóa Athena query & database

X. Estimation

⏱ Time: 6 - 8 hours

💲 Cost: 3$ - 8$

XI. Conclusion

Giải pháp serverless kết hợp AI giúp tăng hiệu quả, giảm chi phí và mở rộng tốt. Phù hợp cho hệ thống bảo mật web hiện đại trên cloud.
